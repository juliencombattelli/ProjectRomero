To install the app, copy the app-debug.apk on a phone with Android 6.0 Marshmallow or higher, and open the file, the app will install automatically

To modify the app, download the repository
adwnload and install Android Studio
Launch android Studio and choose: "Open existing Android Studio project"
Navigate to AndroidRemote -> connect and open

There are two java files and associted xml layouts for 2 activities:
- scan activity to scan nearby BLE device
- remote acticity to connect to a BLE device and control it
